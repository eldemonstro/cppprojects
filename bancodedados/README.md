# Simulação de Banco De Dados

Implementado:
* Banco de dados em struct de até 15 pessoas
* Inserção de valores
* Verificação de valores
* Ordenação por idade (sequencial)
* Pesquisa por idade
* Limpa-banco
* _Debug_ Inserção de uma lista pré definida de pessoas
