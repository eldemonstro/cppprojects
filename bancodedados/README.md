# Simulação de Banco De Dados

Implementado:
* Banco de dados em struct de até 100 pessoas
* Inserção de valores
* Verificação de valores
* Ordenação por idade
* Ordenação por nome
* Pesquisa de idade (sequencial)
* Pesquisa de idade (binario)
* Pesquisa por nome
* Limpa-banco
* _Debug_ Inserção de uma lista pré definida de pessoas
